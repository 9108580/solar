# תיקיית הפרויקט: הצעות מחיר

הפרויקט הועבר מ-`solar-final` ל-`הצעות מחיר`.

אם עדיין קיימת התיקייה הישנה `c:\Users\GHOST\solar-final`:

1. סגרו את Cursor / טרמינלים שפתוחים עליה.
2. מחקו את `solar-final` (או השאירו גיבוי זמני).
3. פתחו ב-Cursor את **`c:\Users\GHOST\הצעות מחיר`** בלבד.

העותק העדכני והפריסה ל-[mes.bet](https://www.mes.bet/) נמצאים כאן.
